*The program will automatically find and press the confirmation 
button after the queue in Aternos.*

## Getting started
##### Installing package
+ **Run your server**
+ **Launch _cmd_ (Windows) or _Terminal_ (MacOs)**
##### Running package
+ **Enter:** 
```python3 auto-aternos.py```
+ **Initialize program with ```start``` command**
+ **Good game to you**
